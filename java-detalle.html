¿Qué es Java?

"Escribe una vez, ejecútalo en cualquier lugar." — Filosofía Java

Java es un lenguaje de programación orientado a objetos, fuertemente tipado y de propósito general. Fue creado por Sun Microsystems en 1995 y hoy es mantenido por Oracle. Es el lenguaje más usado en entornos empresariales a nivel mundial.

Gracias a la JVM (Java Virtual Machine), un programa compilado en Java corre en cualquier sistema operativo sin necesidad de recompilarse. Esto lo hace ideal para sistemas distribuidos y aplicaciones de gran escala.

¿Para qué se usa Java?

  1. Backend empresarial: Bancos, aseguradoras, ERP.
  2. Android: Base histórica del desarrollo móvil.
  3. Microservicios: Spring Boot, Quarkus.
  4. Big Data: Hadoop y herramientas de procesamiento.

Código que aprenderás a escribir

Cada lección en CodeStart está orientada a construir algo real. Aquí un vistazo de lo que verás en el track de Java.

Lección 1: Hola Mundo y estructura básica

  • class — Todo en Java vive dentro de una clase.
  • main — Punto de entrada de cualquier programa.
  • System.out.println — Imprime texto en consola.
public class HolaMundo {
    public static void main(String[] args) {
        System.out.println("¡Hola, CodeStart!");
        System.out.println("Bienvenido a Java 2026.");
    }
}

Lección 2: Variables y tipos de datos

  • int — Números enteros: edad, cantidad, índices.
  • double — Números decimales: precios, temperaturas.
  • String — Cadenas de texto (objeto en Java).
  • boolean — Verdadero o falso para lógica.
public class Variables {
    public static void main(String[] args) {
        int edad = 22;
        double precio = 199.99;
        String nombre = "CodeStart";
        boolean activo = true;

        System.out.println(nombre + " v" + edad);
        System.out.println("Precio: $" + precio);
        System.out.println("¿Activo? " + activo);
    }
}

Lección 3: Tu primera Clase y Objeto (POO)

  • Clase — Molde que define atributos y comportamiento.
  • Objeto — Instancia concreta de una clase.
  • Constructor — Método especial para crear objetos.
public class Estudiante {
    String nombre;
    int nivel;

    // Constructor
    Estudiante(String nombre, int nivel) {
        this.nombre = nombre;
        this.nivel = nivel;
    }

    void presentarse() {
        System.out.println("Hola, soy " + nombre
            + " - Nivel " + nivel);
    }

    public static void main(String[] args) {
        Estudiante dev = new Estudiante("Ana", 3);
        dev.presentarse();
        // Output: Hola, soy Ana - Nivel 3
    }
}
