Python desde cero (Guía Técnica)

Con estas 4 lecciones ya puedes escribir programas que guardan datos, operan con ellos, toman decisiones y se comunican con el usuario.

Nivel 1: Variables y Tipos de Datos

En Python no necesitas declarar el tipo, él lo detecta solo:

  • int: Números enteros. Ej: edad = 25
  • float: Números decimales. Ej: precio = 9.99
  • str: Texto entre comillas. Ej: nombre = "Ana"
  • bool: Verdadero o falso. Ej: activo = True
nombre  = "CodeStart"
version = 1
precio  = 9.99
activo  = True

print(nombre)          # CodeStart
print(type(version))   # <class 'int'>

Nivel 2: Operadores

Python maneja operadores aritméticos, de comparación y lógicos:

# Aritméticos
suma     = 10 + 5    # 15
division = 10 / 4    # 2.5
potencia = 2 ** 8    # 256
modulo   = 10 % 3    # 1

# Comparación (devuelven True o False)
10 > 5    # True
10 == 10  # True

# Lógicos
True and False  # False
True or  False  # True
not True        # False

Nivel 3: Condicionales

Permiten que el programa tome decisiones según una condición:

edad = 18

if edad >= 18:
    print("Eres mayor de edad")
elif edad >= 13:
    print("Eres adolescente")
else:
    print("Eres menor de edad")
  • if: Se ejecuta si la condición es verdadera.
  • elif: Se evalúa si el if anterior fue falso.
  • else: Se ejecuta si ninguna condición se cumplió.

Nivel 4: Input del usuario

Python puede pedirle datos al usuario con input():

nombre = input("¿Cómo te llamas? ")
print("Hola,", nombre)

edad = int(input("¿Cuántos años tienes? "))

if edad >= 18:
    print("Puedes acceder al curso avanzado")
else:
    print("Te recomendamos el curso básico")

Ojo: input() siempre devuelve texto. Si necesitas un número, convierte con int() o float().
